# Tournament system for chess players
Microservice based application for chess tournaments.

	
## 💻 Technologies
Project is created with:
* ASP.NET 6
* Entityframework 6
* REST
* SwaggerUI
* RabbitMQ + MassTransit
* Angular
* Typescript
* Bootstrap
* Scss
